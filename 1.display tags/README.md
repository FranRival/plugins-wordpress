# Post Tags Display Booster 2

Plugin ligero para mostrar las etiquetas (tags) del post con estilo visual color neón, tanto debajo del título como al final del contenido.

## ✨ Features

- Muestra etiquetas debajo del título del post
- Muestra etiquetas al final del contenido
- Colores dinámicos tipo neón
- Incluye contador de uso por etiqueta
- Diseño limpio y llamativo
- Sin configuraciones necesarias (plug & play)

## ⚙️ Cómo funciona

El plugin:

1. Detecta si estás en un post individual (`is_singular('post')`)
2. Obtiene las etiquetas del post actual
3. Genera badges con colores rotativos
4. Inserta automáticamente:
   - Debajo del título (`the_title`)
   - Al final del contenido (`the_content`)

## 🎨 Estilo

Incluye estilos CSS embebidos con efecto glow:

- Bordes redondeados
- Sombras tipo neón
- Colores vibrantes rotativos

## 📦 Instalación

1. Sube el archivo del plugin a:

/wp-content/plugins/post-tags-display-booster-2/  

2. Actívalo desde el panel de WordPress
3. Listo 🚀

## 🧠 Notas

- Solo se aplica al post principal (evita loops secundarios)
- No afecta páginas (`pages`) ni otros post types
- No requiere configuración adicional
